Política de Privacidade

Última atualização: 28 de março de 2026

Esta política descreve como o Atendente24h coleta, usa e protege seus dados pessoais, em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei 13.709/2018).

1. Dados que Coletamos

Coletamos apenas os dados necessários para prestação do serviço:

  • Dados de cadastro: nome, e-mail, telefone e dados da empresa
  • Dados de pagamento: processados de forma segura pelo gateway de pagamento (não armazenamos dados de cartão)
  • Dados de uso: logs de mensagens processadas, configurações do atendente
  • Dados dos clientes finais: mensagens recebidas e enviadas pelo WhatsApp do contratante

2. Como Usamos os Dados

  • Prestar e melhorar o serviço de atendimento automático
  • Processar pagamentos e emitir cobranças
  • Enviar comunicações sobre o serviço (novas funcionalidades, atualizações importantes)
  • Cumprir obrigações legais

3. Compartilhamento de Dados

Não vendemos seus dados. Compartilhamos apenas com:

  • Processadores de pagamento (para cobranças)
  • Provedores de infraestrutura de nuvem (armazenamento seguro)
  • Autoridades quando exigido por lei

4. Retenção de Dados

Mantemos seus dados enquanto sua conta estiver ativa. Após cancelamento, os dados são excluídos em até 90 dias, exceto quando há obrigação legal de retenção.

5. Seus Direitos (LGPD)

Você tem direito a:

  • Acessar seus dados pessoais
  • Corrigir dados incorretos
  • Solicitar exclusão dos dados
  • Portabilidade dos dados
  • Revogar consentimento

Para exercer esses direitos: contato@atendente24h.com

6. Segurança

Utilizamos HTTPS em todas as comunicações, criptografia em repouso para dados sensíveis e controles de acesso rigorosos.

7. Cookies

Utilizamos apenas cookies essenciais para funcionamento do painel administrativo. Não utilizamos cookies de rastreamento de terceiros.

8. Contato do DPO

Encarregado de Proteção de Dados: contato@atendente24h.com
